# hierarchical-rag-multimodal-music-qa
Hierarchical RAG system for multimodal music question–answering using vector-based semantic retrieval and a local LLM.

This repository accompanies the paper *"Hierarchical Retrieval-Augmented Generation for Multimodal Music Question–Answering Using Vector-Based Semantic Retrieval"*.

We present a lightweight system that enables natural language interaction with complex multimodal music datasets. The approach combines multimodal feature extraction with hierarchical, vector-based retrieval and a locally deployed LLM (via Jan) to support semantic querying across heterogeneous data sources.

The system is evaluated through cross-modal retrieval and dataset summarization tasks, demonstrating its ability to connect information across modalities, generate contextually appropriate responses, and surface latent relationships within large collections.

Our results suggest that hierarchical RAG can support higher-level reasoning about dataset structure and content, while significantly reducing the technical barriers associated with traditional file-based or metadata-driven exploration.
